1. 例题[POJ-3176](https://vjudge.net/problem/POJ-3176) ，AC[源码](./POJ-3176.cc)

   题意：给一个数字三角形，问从top走到底部能获得的最大分数是多少？只能沿着两对角线向下走。

   题解：定义dp\[i][j]为从顶部想下走，走到第i层j列的最优分数。一层一层的往下走，每一层记录所有可能的值，然后去更新下一层，最后对底部所有可能的值取max就是最终的答案。

   时间复杂度：O(n^2)
